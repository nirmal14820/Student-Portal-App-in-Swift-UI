
# Student Portal App (SwiftUI)

A simple Student Portal application developed using SwiftUI. This app helps students manage their daily academic activities through an easy-to-use interface.

## Features

* Student Login
* Class Schedule
* Notes Management
* Add, Edit and Pin Notes
* Bus Tracking
* Attendance Tracking
* Notice Board
* Student Profile
* Logout Functionality

## Screens

* Login Screen
* Class Schedule Screen
* Notes Screen
* Note Details & Edit Screen
* Add New Note Screen
* Bus Tracking Screen
* Attendance Screen
* Notice Board Screen
* Profile Screen

## Technologies Used

* Swift
* SwiftUI
* Xcode
* iOS Simulator

## How to Run

1. Download the ZIP file of the project.
2. Extract the ZIP file on macOS.
3. Open the extracted folder.
4. Right-click on the **Studentportal** folder (Do NOT open the **__MACOSX** folder).
5. Select **Open With → Xcode**.
6. Xcode will open the complete project with all files automatically.
7. Select an iPhone Simulator or connect a physical iPhone.
8. Click the **▶ Play (Run)** button in Xcode.
9. The application will build and run successfully.
